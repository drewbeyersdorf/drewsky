   - Pitch: 0 (neutral)

5. **Input script**:
   - Open TUTORIAL_NARRATION_SCRIPT.md
   - Copy Scene 1 (Introduction)

## üí° Quick Tips for Success

### For Voice Narration:
- **Pace**: Read naturally, don't rush
- **Pauses**: Use commas for natural breathing
- **Emphasis**: Mark key words in your TTS tool
- **Test**: Listen to first scene, adjust before doing all

### For Animation:
- **Keep it simple**: Clarity beats flashiness
- **Consistent style**: Same character design throughout
- **Readable text**: Large fonts, high contrast
- **Smooth transitions**: Don't make viewers dizzy

### For Engagement:
- **Start strong**: Hook viewers in first 10 seconds
- **Show value quickly**: Mention "23% fewer hallucinations" early
- **Use real examples**: Actual code, real scenarios
- **End with action**: Clear "Download now" CTA
