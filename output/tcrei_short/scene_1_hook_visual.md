# Scene 1 Hook

**Duration:** 0:00-0:03
**Narration:** STOP letting AI guess what you want!

---

## Scene 1: Visual Specification (0:00-0:03) - "STOP letting AI guess!"

**Overall Goal:** To immediately grab the viewer's attention with a bold, urgent message about the problem.

**Platform:** YouTube Shorts / TikTok / Instagram Reels

**Tool:** Canva Free

**1. Background:**

*   **Color/Gradient:**
    *   Top Half: Gradient from `#FF4136` (red) to `#DC143C` (crimson red). This creates a sense of urgency and warning.
    *   Bottom Half: Solid `#222222` (dark grey) to provide contrast and readability for text.
*   **Mood/Atmosphere:** Urgent, alarming, attention-grabbing, problem-focused.

**2. Main Elements:**

*   **Text Overlay:**
    *   Line 1: "STOP" - Large, impactful, positioned in the top third.
    *   Line 2: "letting AI" - Medium, positioned slightly below "STOP", contrasting color.
    *   Line 3: "guess what you want!" - Medium, positioned in the bottom third.
*   **Characters/Icons:**
    *   Top Left: Canva Element - "Confused Emoji" (search: "confused face emoji") - Style: Outlined, Red (#FF4136), Small, positioned next to "STOP".
    *   Top Right: Canva Element - "Question Mark" (search: "question mark icon") - Style: Filled, White, Small, positioned next to "STOP".
*   **Graphics:**
    *   Dividing Line: A thick, angled, white line (approx. 8pt thickness) visually separating the red and grey background halves. The angle should be slightly tilted upwards from left to right. Think of a "slash" symbol `/`.

**3. Layout:**

*   **Element Positioning:**
    *   **Top Third:** "STOP" and associated icons.
    *   **Center:** "letting AI"
    *   **Bottom Third:** "guess what you want!"
*   **Visual Hierarchy:**
    1.  "STOP" (largest text, brightest color)
    2.  Emoji & Question Mark (visual interest)
    3.  "letting AI" (contrasting color, draws the eye)
    4.  "guess what you want!" (completes the message)

**4. Animations:**

*   **"STOP":**
    *   Entry Effect: Scale In (small to large) - Fast (0.3 seconds).
    *   During-Scene Animation: Pulse - Subtle, keeps the word alive.
*   **Emoji & Question Mark:**
    *   Entry Effect: Fade In (0.5 seconds).
    *   During-Scene Animation: Slight Jiggle (subtle, adds movement).
*   **"letting AI":**
    *   Entry Effect: Slide In from Left (0.5 seconds).
*   **"guess what you want!":**
    *   Entry Effect: Slide In from Right (0.5 seconds).
*   **Dividing Line:**
    *   Entry Effect: Wipe (Left to Right) (0.3 seconds)
*   **Timing:**
    *   0:00-0:00.3: "STOP" Scale In + Dividing Line Wipe
    *   0:00.3-0:00.8: Emoji & Question Mark Fade In
    *   0:00.8-0:01.3: "letting AI" Slide In
    *   0:01.3-0:01.8: "guess what you want!" Slide In
    *   0:01.8-0:03:  All elements remain on screen with subtle animations.

**5. Typography:**

*   **"STOP":**
    *   Font: "Bebas Neue" (Bold) - readily available in Canva
    *   Size: 90-100pt (adjust based on screen legibility)
    *   Color: White (#FFFFFF)
*   **"letting AI":**
    *   Font: "Montserrat" (SemiBold)
    *   Size: 60-70pt
    *   Color: `#FF4136` (Red)
*   **"guess what you want!":**
    *   Font: "Montserrat" (Regular)
    *   Size: 50-60pt
    *   Color: White (#FFFFFF)
    *   Line Height: Adjust to ensure readability on smaller screens.

**Example Canva Implementation:**

1.  Set canvas size to 1080 x 1920 px (9:16).
2.  Create the gradient background using Canva's background tool.
3.  Add a rectangle shape, cover half the screen and set the color to #222222.
4.  Add the text elements using the specified fonts, sizes, and colors.
5.  Find the "Confused Emoji" and "Question Mark Icon" in the Canva Elements library and add them to the design.
6.  Add a line element. Rotate and resize it to create the angled dividing line. Set the weight to 8pt and color to White.
7.  Animate each element individually using Canva's animation features, following the timing and effects described above.
8.  Download as an MP4 video.

This detailed specification will allow you to easily create a visually compelling and attention-grabbing hook for your vertical video using Canva. The bold colors, clear typography, and subtle animations will ensure that your message is seen and understood on mobile devices.
