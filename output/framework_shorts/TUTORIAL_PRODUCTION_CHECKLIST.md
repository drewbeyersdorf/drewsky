## Recording Checklist

### Before Recording
- [ ] Close all unnecessary apps
- [ ] Turn off notifications (Do Not Disturb mode)
- [ ] Clear browser tabs except demo tabs
- [ ] Check audio levels (test recording 30 seconds)
- [ ] Check camera focus and framing
- [ ] Review first 3 slides to get into rhythm
- [ ] Have water and script notes nearby

### Recording Structure

**INTRO (0:00-0:30)**
- [ ] Scene: Face cam full screen
- [ ] Energy: High, hook them immediately
- [ ] Record 2-3 takes, pick best

**SECTIONS 1-4 (0:30-8:00)**
- [ ] Scene: Slide + face cam corner
- [ ] For each slide:
  - [ ] Present slide content
  - [ ] Switch to full screen for live demos
  - [ ] Return to slide + face cam
- [ ] Record in sections (can edit together)
- [ ] OK to re-record individual slides

**SECTION 5 - Live Build (8:00-10:00)**
- [ ] Scene: Full screen ChatGPT/Claude
- [ ] Picture-in-picture face cam
- [ ] Type prompt live (or pre-type and reveal)
- [ ] Show actual results
- [ ] Can speed up typing in edit if too slow

**OUTRO (10:00-10:30)**
- [ ] Scene: Face cam full screen
- [ ] Call to action: Like, Subscribe, Download cheatsheet
- [ ] End on high energy

## During Recording - Pro Tips

### Pacing
- Speak 10% slower than normal
- Pause between major points (easier to edit)
- Don't rush through code examples
- Allow 2-3 seconds silence between sections

### Energy
- Stand while recording (more energetic)
- Use hand gestures (even if not fully visible)
- Smile when appropriate
- Vary vocal tone to maintain interest

### Technical
- Save recording every 15 minutes
- If you mess up, pause, then restart that sentence
- Leave editing markers: "Edit point" or clap hands
- Record room tone (30 seconds silence) for audio editing
